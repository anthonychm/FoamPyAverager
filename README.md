## ⚙️ Installation
This package was developed with Python 3.13 and the following libraries:
- matplotlib 3.10.8
- numpy 2.4.2
- scikit_learn 1.8.0

To install:
```python
git clone https://github.com/anthonychm/FoamPyAverager.git
cd FoamPyAverager
pip install -r requirements.txt
```
